#California Housing Price Prediction

## Project Overview
This project predicts California housing prices using Machine Learning Regression algorithms. It includes data preprocessing, feature scaling, model training, evaluation, and house price prediction.

## Dataset
- California Housing Dataset
- File: `housing.csv`

##  Machine Learning Workflow
1. Load Dataset
2. Data Preprocessing
3. Handle Missing Values
4. Feature Scaling
5. Train-Test Split
6. Model Training
7. Model Evaluation
8. House Price Prediction

##  Machine Learning Models Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Gradient Boosting Regressor

##  Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

##  Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

1. Clone this repository.
